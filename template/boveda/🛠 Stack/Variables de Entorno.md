## Variables opcionales útiles

| Variable | Propósito |
|----------|-----------|
| `SUPABASE_ACCESS_TOKEN` | Personal Access Token para Management API (migraciones DDL, admin de proyecto) |
| `DATABASE_URL` | URL de conexión directa a PostgreSQL (para `psql`, `pg_dump`, migraciones raw) |
